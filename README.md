# OtakuList

Aplicación web para gestionar tu lista personal de anime y manga.
Construida con React + TypeScript + Tailwind CSS.

## Tecnologías
- React + TypeScript
- Tailwind CSS
- React Router
- Jikan API (datos de anime y manga)
- LocalStorage (persistencia de datos)
